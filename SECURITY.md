# Security Policy

## Supported Versions
Only the latest version of urBackend is currently supported for security updates.

## Reporting a Vulnerability
**Please do not report security vulnerabilities through public GitHub issues.**

If you discover a security vulnerability within urBackend, please send an email to [yashpouranik124@gmail.com] or open a private advisory on GitHub. We will acknowledge your report within 48 hours and provide a timeline for a fix.
